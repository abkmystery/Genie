# Roadmap

## Phase 1

- Launcher, panel, ingestion, grounded answers, traces, and profile switching.
- Phase 1.5 adds explicit activity recording, local speech-provider diagnostics, and session-based temporal summaries.
- Guided Task mode adds on-screen arrow guidance, target grounding, and manual step-by-step help without autonomous control.

## Phase 2 Candidates

- URL source loader.
- Database source loader.
- Form detection.
- Stronger always-on voice experiences and wake word support.
- Richer desktop event capture and finer-grained action inference.
- Approval-aware action planning.
- Browser, file, email, and command tools.
- Better semantic retrieval and embeddings.
- Stronger multimodal understanding and OCR integration.
