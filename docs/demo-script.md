# Demo Script

Target length: 2 to 3 minutes.

## 1. Open With Trust

Launch Genie from the packaged app. Open Settings briefly and show the status pill: `Live Gemma`.

Narration: "Genie is a privacy-first desktop companion. The app talks to its local backend, and Demo mode routes model calls through the demo gateway so provider credentials are not shown in the UI."

## 2. Screen Understanding

Open a real page, then ask: `What am I looking at and what should I do next?`

Show the answer is concise and screen-grounded.

## 3. Private File Grounding

Attach a small PDF, DOCX, XLSX, CSV, TXT, or image. Ask a question whose answer is inside the file.

Show citations and the debug trace id.

## 4. Guided Task Mode

Ask: `Guide me through checking high prize competitions on Kaggle`.

Show the task card and overlay arrow. Emphasize that Genie does not click or type; the user stays in control.

## 5. Local-First Option

Open Settings and show Local profile fields for endpoint and model. If the local runner is available, switch to it and run diagnostics.

## Rehearsal Rule

Run the script three times before recording. The final recording should not show fallback/mock responses unless you explicitly frame that as offline resilience.
